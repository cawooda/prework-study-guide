Prework Study Guide

HTML

  • Head section contains non visible code comprising the meta tags as well as link tags, style and script tags
  • UTC-18 is the character set for the document
  • An element describes the whole element including content, a tag is just the <>
  • The body contains the visual content and tags structuring the document
  • Id is how the tag is identified within the doucment.
  • Class is similar to id in the way it is used so far but has some extensions

Git

  • git status gets the status of the repo including what branch you are on
  • git checkout creates a new branch which you name by git checkout -b NAME
  • git pull origin main gets the latest version of the remote repository onto the local system
  • git push origin branch-name sends to the remote as a branch
  • git commit -m followed by a message labesl the commit
  • git add -A adds all files to the commit staging area

CSS

  • A margin indicates how much space resides outside The elementprovideing a buffer with other elements.
  • Padding indicates the space within an element between its content and the border.
  • a class selector selects all elements that have that class applied to them.
  • a CSS declaration contains the selector and the property/value pairs
